# My Subnautica Mods

Welcome to the **My Subnautica Mods** repository! This repo contains a collection of mods developed and maintained by [@slin-o](https://github.com/slin-o) for the popular game Subnautica.

## Overview

This repository features a variety of mods designed to enhance and extend your Subnautica gameplay experience. Each mod aims to add new features, improve gameplay mechanics, or fix existing issues within the game.

## Features

- ⚡️ Multiple community-driven mods in a single repo
- 🛠️ Open source for modification and contribution
- 🐟 Compatible with modern Subnautica builds
- 🔧 Easy installation & modular approach

## Requirements

- **Subnautica** version 2.0 or newer
- **BepInEx**
- **Nautilus**

## Mods Included

- **AccessStorageModules**  
  Enables streamlined access to storage modules within your vehicles, making inventory management easier and more intuitive.

- **DroneInventory**  
  Adds drone-based inventory management, allowing you to transfer items remotely and automate resource transport with deployable drones.

- **PDARenameBeacon**  
  Lets you rename beacons directly from your PDA, for smarter navigation and organization in the game world.

## Installation

1. Download the desired mod(s) from [Releases](https://github.com/slin-o/my-subnautica-mods/releases) or clone this repository.
2. Ensure [BepInEx](https://bepinex.github.io/) and [Nautilus](https://github.com/SubnauticaModding/Nautilus) are installed for Subnautica 2.0+.
3. Copy the mod folder(s) into your Subnautica **BepInEx/plugins** directory:
    ```
    C:\Program Files (x86)\Steam\steamapps\common\Subnautica\BepInEx\plugins
    ```
4. Launch Subnautica. Mods should be loaded automatically.
